# API接口文档

## 基础URL

- 本地开发: `http://127.0.0.1:5000`
- 线上访问: https://flask-next-demo.vercel.app

---

## 接口1：首页欢迎接口

### 请求信息

- **地址**: `/api/index`
- **方法**: `GET`
- **参数**: 无

### 返回示例

```json
{
  "msg": "欢迎使用社区便民物品借用管理平台"
}
```

## 接口2：查询所有闲置物品

### 请求信息

- **地址**: `/api/goods`
- **方法**: `GET`
- **参数**: 无

```json
{
  "code": 200,
  "data": [
    {
      "id": 1,
      "name": "冲击钻",
      "category": "维修工具",
      "price_per_day": 3.0

    {
      "id": 2,
      "name": "投影仪",
      "category": "影音家电",
      "price_per_day": 8.0
    }
  ],
  "msg": "物品查询成功"
}
```

## 接口3：新增闲置物品

### 请求信息

- **地址**: `/api/add_goods`
- **方法**: `POST`
- **Content-Type**: `application/json`

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| name | string | 是 | 物品名称 |
| category | string | 否 | 物品分类（工具/家电/户外等） |
| price_per_day | number | 是 | 单日租赁价格 |

### 请求示例

```json
{
  "name": "露营帐篷",
  "category": "户外用品",
  "price_per_day": 5.0
}
```

### 返回示例

**成功** (200):
```json
{
  "code": 200,
  "msg": "闲置物品发布成功"
}
```

**失败** (400):
```json
{
  "code": 400,
  "msg": "物品名称、单日租赁价格不能为空"
}
```

## 接口4：租赁费用计算

### 请求信息

- **地址**: `/api/calc_rent`
- **方法**: `GET`

### 请求参数（Query）

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| day | number | 是 | 借用天数 |
| price | number | 是 | 单日租金 |

### 请求示例

```
GET /api/calc_rent?day=3&price=5
```

### 返回示例

**成功** (200):
```json
{
  "code": 200,
  "total_rent": 15.0,
  "msg": "费用计算完成"
}
```

**失败** (400):
```json
{
  "code": 400,
  "msg": "天数、单价必须输入数字"
}
```
